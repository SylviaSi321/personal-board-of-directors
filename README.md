# 🪑 人生私董会 Personal Board of Directors

> 用 AI 为自己组一支私人智囊团 — 6 位不同维度的私董成员 + 1 位状态驱动的教练，通过结构化深度对话，帮你看见自己还没看见的。

**先提问后建议，先看清问题再寻找答案。**
**案主的状态是指南针，流程是工具，不是目的。**

## ✨ 这是什么？

「人生私董会」是一个 **AI Prompt 项目**，把真实私董会的议事流程搬到了 AI 对话中。

你只需要把 Prompt 粘贴到 ChatGPT / Claude / Gemini / 扣子 等任何主流 AI 平台，就能获得一场有结构、有深度的多角色决策对话体验。

**它不替你做决定，但会帮你看清全貌。**

### 核心特点

- 🎯 **状态驱动的教练** — 教练不看流程的钟，看你的状态，三层能力模型（感知·调节·推进）
- ⚡ **提问与建议严格分离** — 先用提问帮你看清，再用建议帮你找方向
- 🔍 **重新定义根本问题** — 你带着「问题 A」进来，发现真正的问题是「问题 B」
- 🫁 **弹性深挖-收束节奏** — 没有固定轮次限制，像呼吸一样交替深挖与整理
- 🔀 **语气自适应** — 情绪话题温柔，决策话题理性

## 📋 五阶段流程

```
阶段 1: 认识你 + 聚焦问题
  └→ 教练重构为「我如何___？」→ 你确认

阶段 2: 探究提问 ⭐ 最有价值的环节
  └→ 成员轮流提问，你来回答
  └→ 只提问，不给建议
  └→ ⭐v3.1 教练根据你的状态动态调整节奏
  └→ 弹性轮次，按你的节奏来

阶段 3: 重新定义根本问题
  └→ 每位成员写下「我认为根本问题是___」
  └→ ⭐v3.1 准备度检查后，你选择最触动的一条

阶段 4: 建议交锋
  └→ 围绕根本问题给建议
  └→ 成员之间可以互相碰撞

阶段 5: 收束
  └→ 自适应问题演进回顾 + 行动清单 + 教练寄语
  └→ ⭐v3.1 收束 = 标记你走到的位置，而非强行给结论
```

## 🚀 快速开始

### 方式一：在线体验（推荐）

👉 **[打开展示页](https://你的用户名.github.io/personal-board-of-directors/)** — 选择 AI 平台，一键复制 Prompt 开始

1. 打开 [`prompt.md`](./prompt.md) 文件

4. 发送，按教练引导开始

## 🧬 设计溯源

本 Prompt 融合了三大来源：

| 来源 | 贡献 |
|------|------|
| 李继刚「圆桌讨论」V1 | 主持人驱动制、动态邀请人物、用户控制指令、ASCII 思考框架 |
| 李继刚「圆桌讨论」V2 | 揭示偏见、费曼简化、德鲁克行动聚焦 |
| 真实私董会流程 | **提问与建议分离**、**重新定义根本问题**、**案主一问一答**、**案主反馈** |

### v3.1 核心升级（对比 v3.0）

| 变化 | v3.0 | v3.1 |
|------|------|------|
| 教练定位 | 流程守护者 | **状态驱动的守护者**，三层能力模型 |
| 提问轮次 | 固定 2-3 轮 | **弹性轮次**，基于案主状态信号 |
| 每轮结束 | 直接给选项 | **先镜像反馈 + 探测感受**，再给选项 |
| 收束判断 | 按轮次 | **基于信号累积**三维检测 |
| 阶段切换 | 直接切换 | **准备度检查** |
| 收束纪要 | 假设走完全程 | **自适应纪要** |

### v3.0 核心升级（对比 v2.0）

| 变化 | v2.0 | v3.0 |
|------|------|------|
| 教练角色 | 请人 + 提问 | **流程守护者**，严格控制阶段切换 |
| 讨论结构 | 成员一次性全部发言 | **提问轮和建议轮严格分开** |
| 案主参与 | 被动听 | **一问一答，深度参与提问轮** |
| 根本问题 | 无 | **新增：成员各自定义，案主选择** |
