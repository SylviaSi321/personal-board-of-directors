🔄 五阶段对话流程

提问与建议严格分离——这是真实私董会最重要的规则。先帮你看清问题，再给出建议。

阶段 1
🪞 认识你 + 聚焦问题
教练了解你的背景和议题，重构为「我如何___？」的标准问题，温和揭示可能的偏见。
阶段 2 ⭐
🔍 探究提问
最有价值的环节。成员轮流向你提问，你来回答。只提问、不给建议。帮你和大家看见更深处。
阶段 3 ⭐
💡 重新定义根本问题
每位成员写下「我认为根本问题是___」，你选择最触动你的那一条。往往不同于你最初的描述。
阶段 4
💬 建议交锋
确认根本问题后，成员们才开始给建议。可以互相碰撞、挑战。你可以追问任何人。
阶段 5
🎯 收束行动
问题演进回顾、费曼简化、共识分歧、行动清单、教练寄语。一张圆桌纪要带走。

✨ v3.0 核心设计

四个让这个私董会与众不同的设计。

🎯
教练驱动流程
教练不是专家、不给建议。教练的价值是「控制什么时候该做什么」——该提问时不许给建议，该建议时鼓励交锋。
⚡
提问与建议分离
真实私董会最重要的规则。先用提问帮你看清全貌，再用建议帮你找到方向。大多数人跳过了提问，所以困在了原地。
🔍
重新定义根本问题
私董会最有顿悟感的时刻——你带着「问题A」进来，经过提问后发现，真正的问题其实是「问题B」。
🔀
语气自适应
情绪话题温柔，决策话题理性，方案话题干脆。永远不说「你应该」，而是「你可以考虑」。

📖 怎么开始

选择平台

Step 2
简单介绍自己

🧬 设计溯源

本 Prompt 融合李继刚「圆桌讨论」V1 + V2 + 真实私董会流程。v3.1 核心升级：教练从「流程驱动」→「状态驱动」，三层能力模型，弹性深挖-收束呼吸机制。

设计要素 来源 在 v3.0 中的体现
主持人驱动制 V1 私董会 教练严格控制流程，六大职能（引导/守护/重构/追问/提炼/节奏）
动态邀请人物 V1 根据议题从成员库选择最合适的组合，可随时「换人」
用户控制指令 V1 继续提问 / 进入下一步 / @追问 / 补充 / 换人 / 收束
思考框架可视化 V1 建议轮综述时生成可视化思考结构
揭示偏见 V2 阶段1 问题聚焦时温和揭示预设偏见
费曼简化 V2 阶段5 收束时「一句话说清楚」
德鲁克行动聚焦 V2 阶段5 行动清单（优先级 + 时间 + 难度）
提问与建议分离 私董会 阶段2 vs 阶段4 严格分开，这是最重要的规则
重新定义根本问题 私董会 阶段3 全新环节——最有顿悟感的时刻
案主一问一答 私董会 阶段2 核心交互方式，深度参与
语气自适应 新增 根据议题类型自动调整沟通风格
问题演进回顾 新增 收束时展示：表面问题 → 聚焦问题 → 根本问题 的演进
